# Bitcoin Price Forecasting (LSTM)

End-to-end time-series forecasting workflow using BTC-USD historical prices (via `yfinance`) and an LSTM model.

## What this shows
- **Time-series preprocessing** (windowing / scaling / chronological split)
- **Sequence modeling** with an LSTM
- **Evaluation + visualization** of predicted vs actual values

## Repo structure
- `src/` — code
- `data/` — optional local data notes
- `docs/` — report (Markdown + PDF) and figures
